# My First Project

A simple beginner HTML/CSS/JavaScript project hosted on GitHub.

## 🚀 Features

- Accessible, responsive HTML5 page
- Button click interaction with DOM feedback (no blocking `alert()`)
- Clean, maintainable CSS with proper font-family fallbacks

## 🛠️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/anees1975/my_first_project-.git
   ```
2. Open `index.html` in any web browser — no build step required.

## 📁 Project Structure

```
├── index.html   # Main page
├── style.css    # Styles
├── script.js    # JavaScript behaviour
└── README.md    # This file
```
